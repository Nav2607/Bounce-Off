# Bounce-Off-Algorithm
## About  
This project shows two rectangles moving toward each other and bouncing away. It uses p5.js and p5.play libraries.

## What the Code Does  
- Makes a canvas that is 1200 by 800.  
- Creates two rectangles as sprites.  
- Gives them colors and lets you see their outlines.  
- Moves one rectangle up and the other down.  
- When they touch, they collide and bounce away from each other.
